# 📊 مكتبة المصفوفات والشعاعات (My Matrix Library)

مشروع تعليمي بلغة Python لبناء مكتبة مبسطة للجبر الخطي من الصفر، تشمل التعامل مع **الشعاعات (Vectors)** و**المصفوفات (Matrices)** وتنفيذ العمليات الرياضية الأساسية عليها.

---

## 🎯 هدف المشروع

يهدف هذا المشروع إلى:

* فهم أساسيات الجبر الخطي برمجيًا
* تطبيق مفاهيم البرمجة كائنية التوجه (OOP)
* استخدام الدوال السحرية في Python مثل `__add__`, `__mul__`
* بناء مكتبة مشابهة بشكل مبسط لمكتبات مثل NumPy

## 🧱 مكونات المشروع

### 🔹 1. Sho3a3 (الشعاع)

يمثل قائمة من الأرقام، ويدعم العمليات التالية:

* الطول: `len(s)`
* الفهرسة: `s[0]`
* التكرار: `for x in s`
* الجمع والطرح:

  ```python
  a + b
  a - b
  ```
* الضرب:

  * شعاع × شعاع → جداء نقطي
  * شعاع × رقم → ضرب عنصر-بعنصر
* القسمة والأس (اختياري)
* دوال مساعدة:

  * `sum()`
  * `shape()`

---

### 🔹 2. Masfofah (المصفوفة)

تمثل مجموعة من الشعاعات (صفوف):

* معرفة الشكل: `shape()`
* الفهرسة: `m[0]`
* الجمع والطرح بين مصفوفتين
* ضرب المصفوفات
* استخراج الأعمدة
* المنقول (Transpose)
* المجموع:

  * `axis=0` (أعمدة)
  * `axis=1` (صفوف)
* الإضافة (صفوف أو أعمدة)

---

### 🔹 3. operations.py

يحتوي على دوال مساعدة:

* إنشاء مصفوفة أصفار:

  ```python
  masfofah_asfar(rows, cols)
  ```
* إنشاء مصفوفة واحدات:

  ```python
  masfofah_wahidat(rows, cols)
  ```
* إنشاء مصفوفة من قائمة:

  ```python
  i9n3_masfofatan(data)
  ```
* مصفوفة الوحدة:

  ```python
  identity_masfofah(n)
  ```
* الجداء العنصري:

  ```python
  elementwise_product(A, B)
  ```
